# EDA-of-NYPD-Shooting-Incidents-Reports
Explorative Data Analysis of the NYPD Shooting Incident Reports data set. Examined trends in shooting and homicide incidents over time, analyzed demographic disparities in victims age and race, and built a predictive logistic regression model.
Libraries used: 
- Tidyverse
- Lubridate
- Dplyr
- Knitr
- tidymodels
